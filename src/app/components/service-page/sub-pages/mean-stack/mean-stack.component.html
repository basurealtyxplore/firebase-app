What is MEAN?

  • MEAN is an Acronym of Mongo DB, Express JS, Angular and Node JS.
  • All composite of Javascript which makes them to stand apart in other frameworks.
  • One Language for all Server, Client and Databse.
  • Your application database is stored in Mongo DB which is NoSQL Database.
  • Your application APIs are created in Express JS.
  • Your application server configuration and networks are configured in Node JS
  • Your application front end framework would be blazing fast Angular.
